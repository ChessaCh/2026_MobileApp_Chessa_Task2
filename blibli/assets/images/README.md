# Assets Directory

This folder is for image assets used in the Blibli Login App.

## Required Images:

1. **blibli_logo.png** - The Blibli logo (approximately 200x50px)
   - Should be a PNG with transparent background
   - Place the official Blibli logo here

2. **login_illustration.png** - The login page illustration (approximately 800x600px)
   - The colorful illustration with mascot characters
   - Should include the festive/celebration theme

## Notes:
- If these images are not present, the app will show placeholder icons instead
- For production use, make sure to use properly licensed images
- Recommended formats: PNG with transparency
